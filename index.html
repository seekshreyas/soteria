SOTERIA is a project that focuses on exploring and analysing security breach data. For our analysis we will be using a VERIS Community Database. It is a set of metrics designed to provide a common language for describing security incidents in a structured and repeatable manner.

We analysed and extracted interesting features to ascertain defining attributes for security breaches, built an interface for quick filtering of breach data, and finally used a similarity algorithm to identify similar breaches.

NaN = missing values

Unknown = Not Known from the VCDB database
